Hybrid Cryptography Strategy

Transitioning to PQC is risky if new algorithms have undiscovered vulnerabilities. We mitigate this using a Hybrid Key Exchange mechanism.

1. Classical Layer (X25519)

Standard Elliptic Curve Diffie-Hellman. Fast, proven secure against conventional computers. Ensures connection remains secure even if PQC fails.

2. Post-Quantum Layer (Kyber/ML-KEM)

Lattice-based key encapsulation mechanism. Resistant to attacks from future large-scale quantum computers.

Why WolfSSL?

We selected WolfSSL v5.7+ because it offers a native --enable-pqc build flag, integrates seamlessly with Envoy via custom builds, and is optimized for embedded/mobile environments where binary size matters.
